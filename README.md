# Similar World Detector
Detect similar words present in the source using fuzzball with detailed console and extract to excel file

Start with 
=============

    npm install
    
or 
    
    yarn
    
## Syntax

    node parse.js \<from>-\<to>

**Example**

    node parse.js 0-200

*Run parallel processes to make efficient*
